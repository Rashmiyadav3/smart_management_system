# Smart Management System (SMS)

A full-stack web application that provides a centralized platform for managing user interactions through a responsive landing page and a secure admin panel.

---

## Project Overview

The **Smart Management System** allows users to submit contact queries via a public website, while administrators can view and manage these submissions from an admin dashboard.
